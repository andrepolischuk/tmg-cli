# tmg-cli

  > Timer generator

  [tmg][tmg] - API for this module

## Install

```sh
npm install --global tmg-cli
```

## Usage

```sh
tmg --help

  Usage
    tmg [date] [time]

  Examples
    tmg
    tmg 12:15
    tmg 2015.02.11 12:00:00
```
